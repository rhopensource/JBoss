JBoss Demo Environemnt Setup
==========================================================================================================

This tool helps you setup JBoss env in your laptop for a quick demo.

Pre-Requisites:
*** Red Hat Enterprise Linux 7.2+ Workstation on your laptop
    * Create an user account "jboss".
      # useradd jboss; passwd jboss

    * Add user jboss to sudo list.
      # visudo

    * Test jboss user has root access.
      # su - jboss
      $ sudo id    should show you root id. Exit the jboss user shell.

    * Update latest patches in your RHEL OS.
      # yum -y update


*** Postgres database on RHEL 7.2+ for JBoss Operations Network
    * Install Postgres database.
      # yum -y install postgresql\*

    * Initialize Postgres database for first time
      # service postgres initdb
      # service postgres start

*** Postgresql.conf settings to be updated as shown below and Postgres database restarted
    $ sudo su - postgres -c "vi /var/lib/pgsql/data/postgresql.conf"

    shared_buffers = 80MB
    work_mem = 2048MB
    checkpoint_segments = 10
    
    # service postgres restart

*** Postgres database SQL for JBoss Operations Network. Run /tmp/jbosson.sql generated by the tool.
    $ ./jboss.sh -i jonsql
    $ sudo su - postgres -c "psql -f /tmp/jbosson.sql"


*** Postgres database client authentication method needs to be changed from ident to md5 prior to 
    installation of JBoss Operations Network. File to edit is /var/lib/pgsql/data/pg_hba.conf as 
    postgres user.
    $ sudo su - postgres -c "vi /var/lib/pgsql/data/pg_hba.conf"

*** Download the latest JDK and install it as per your preference. 
    Set JAVA_HOME to correct path and add $JAVA_HOME/bin to your PATH
